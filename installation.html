Installation

Prerequisites

Before installing PySynthLab, ensure you have at least the following installed:

  • Python 3.8 or higher

  • Virtualenv recommended

  • Z3 Theorem Prover

Installation Steps

  1. Clone the repository:

    git clone https://github.com/gschandan/PySynthLab.git
cd PySynthLab

  2. Create and activate the virtual environment:

    python -m venv venv
source venv/bin/activate

  3. Install the requirements:

    You have several options to install the requirements:

    1. Using pip without venv:

      pip install -r requirements.txt

    2. Using Make:

      make init

    3. Using pip with venv:

      ./venv/bin/pip install -r requirements.txt

    4. Using Make with venv:

      make init_venv
