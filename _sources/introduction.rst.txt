Introduction to PySynthLab
==========================

PySynthLab is a powerful Python library designed for program synthesis using counterexample-guided inductive synthesis (CEGIS).
It provides a flexible and extensible framework for developing and experimenting with various synthesis strategies.

Key Features
------------

- Multiple candidate generation strategies
- Configurable synthesis parameters
- Integration with Z3 SMT solver
- Extensible architecture for custom strategies

Who should use PySynthLab?
--------------------------

- Researchers in program synthesis
- Developers working on automated programming tools
- Students learning about program synthesis techniques
