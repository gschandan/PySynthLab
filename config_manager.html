Bases: object

Options for configuring the Synthesis process.

Other:

input_source (str): Input source. Choices are STDIN or a path to a problem file.

Note

When using a YAML configuration file, these options can be grouped under ‘logging’, ‘synthesis_parameters’, and ‘solver’ sections. For example:

logging:

level: “DEBUG”

synthesis_parameters:

max_iterations: 20 operation_costs:

‘+’: 1 ‘-’: 1 ‘*’: 3 ‘If’: 4 ‘Neg’: 2

solver:

name: “z3” timeout: 30000
