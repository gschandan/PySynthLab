CandidateGenerator Class

class src.cegis.z3.candidate_generator.candidate_generator_base.CandidateGenerator(problem: SynthesisProblem)

Bases: ABC

Abstract base class for candidate function generators.

This class defines the interface for generating and pruning candidate functions in the context of program synthesis.

problem

The synthesis problem instance.

Type:
