SynthesisProblem

class src.cegis.z3.synthesis_problem.SynthesisProblem(problem: str, options: Options = None)

Bases: object

A class representing a synthesis problem in the SyGuS format.

This class provides methods for parsing, analyzing, and manipulating synthesis problems expressed in the SyGuS (Syntax-Guided Synthesis) format. It includes functionality for converting SyGuS to SMT-LIB format, initializing Z3 variables and functions, parsing constraints, substituting constraints.

Attributes:

logger (logging.Logger): Logger for the class. options (Options): Configuration options for the synthesis problem. input_problem (str): The original input problem in SyGuS format. parser (SygusV1Parser | SygusV2Parser): The parser used to parse the problem. problem (Program): The parsed SyGuS program. symbol_table (SymbolTable): Symbol table for the parsed program. printer (SygusV2ASTPrinter): Printer for the AST. context (SynthesisProblemContext): Context holding various data structures and solvers.

Example:
(synth-fun max2 ((x Int) (y Int)) Int …)””
>>> options = Options()
>>> synthesis_problem = SynthesisProblem(problem_str, options)
>>> print(synthesis_problem.get_logic())
LIA
>>> print(synthesis_problem.get_synth_funcs())
{'max2': FunctionDescriptor(...)}

substitute_candidates(constraints: Collection[ExprRef], candidate_functions: List[tuple[FuncDeclRef, FuncDeclRef | ExprRef | Callable]]) → List[ExprRef]

Substitute candidate expressions into a list of constraints.

This method substitutes both candidate functions and predefined functions into the given constraints.

Args:

constraints (Collection[z3.ExprRef]): The list of constraints to substitute. candidate_functions (List[tuple[z3.FuncDeclRef, typing.Union[z3.FuncDeclRef, z3.ExprRef, Callable]]]):

The candidate functions and their replacements.

Returns:

List[z3.ExprRef]: The list of substituted constraints.

Example:
>>> problem_str = ""(set-logic LIA)

(synth-fun max2 ((x Int) (y Int)) Int) (define-fun min2 ((x Int) (y Int)) Int (ite (<= x y) x y)) (constraint (>= (max2 (min2 a b) c) c))””

>>> synthesis_problem = SynthesisProblem(problem_str)
>>> max2 = synthesis_problem.context.z3_synth_functions['max2']
>>> a, b, c = Int('a'), Int('b'), Int('c')
>>> replacement = lambda x, y: If(x > y, x, y)
>>> candidates = [(max2, replacement)]
>>> substituted = synthesis_problem.substitute_candidates(synthesis_problem.context.z3_constraints, candidates)
>>> print(substituted[0])
If(If(a <= b, a, b) > c, If(a <= b, a, b), c) >= c

Examples

Creating a SynthesisProblem instance:

problem_str = "(set-logic LIA)\n(synth-fun max2 ((x Int) (y Int)) Int)\n(declare-var a Int)\n(declare-var b Int)\n(constraint (>= (max2 a b) a))"
synthesis_problem = SynthesisProblem(problem_str)

Getting the logic of the problem:

logic = synthesis_problem.get_logic()
print(logic)  # Output: LIA

Getting synthesis functions:

synth_funcs = synthesis_problem.get_synth_funcs()
print(list(synth_funcs.keys()))  # Output: ['max2']

Parsing constraints:

synthesis_problem.parse_constraints()
print(synthesis_problem.context.z3_constraints)  # Output: [And(max2(a, b) >= a)]

Substituting candidate expressions:

from z3 import Int, Function
max2 = synthesis_problem.context.z3_synth_functions['max2']
a, b = Int('a'), Int('b')
replacement = lambda x, y: If(x > y, x, y)
substituted = synthesis_problem.substitute_constraints(synthesis_problem.context.z3_constraints, [max2], [replacement])
print(substituted[0])  # Output: If(a > b, a, b) >= a
